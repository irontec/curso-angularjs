### ¿Como le indicamos si debe ser un nombre, un atributo o una clase?

Convención
    - Elemento si tomará el control del template
    - Atributo si "decora" el elemento.

La restricción se define con:
    - 'A' attribute name
    - 'E' element name
    - 'C' class name

require: '?ngModel' or '^ngModel'
